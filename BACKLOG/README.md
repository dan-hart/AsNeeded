# BACKLOG Directory

This directory contains project management files for the AsNeeded iOS app using [Backlog.md](https://github.com/MrLesk/Backlog.md) - a CLI-based project management tool.

## Getting Started

Install with Homebrew:
`brew install backlog-md`

Use Claude (AI assistant) or the browser interface to manage tasks. The backlog tool is fully integrated with Claude for seamless project management.

## Usage

- **With Claude**: Simply ask Claude to create, view, edit, or manage tasks. Claude handles all backlog operations automatically.
- **With Browser**: Use `backlog browser` to open the web interface for visual task management.

## Learn More

- Full CLI documentation: `backlog --help`
- Advanced features: [Backlog.md Repository](https://github.com/MrLesk/Backlog.md)
- Project-specific guidelines: See `CLAUDE.md` in this directory
